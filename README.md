# Fathima Thiraya - Portfolio

A modern, responsive personal portfolio showcasing my work as a Full Stack Developer and QA Automation Engineer.

## About

Hi, I'm Fathima Thiraya, a final-year BSc Management Information Technology student passionate about building quality software. I specialize in QA automation, DevOps practices, and full-stack development.

## Tech Stack

- **Frontend:** React.js, JavaScript, HTML5, CSS3
- **Backend:** Node.js, Express.js, PHP
- **Database:** MongoDB, MySQL
- **Testing:** Playwright, Postman
- **DevOps:** GitHub Actions, CI/CD, Git

- Typing animation effect
- Contact form with email integration
- Animated counters and statistics
